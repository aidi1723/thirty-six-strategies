# Thirty-Six Stratagem Operators

Use these as modern strategic operators. Do not treat them as permission for illegal or unethical behavior; translate them into lawful competitive positioning, negotiation design, content strategy, contract architecture, intelligence analysis, and risk control.

## 胜战计：顺风局扩充势能

Core logic: 避实击虚，转移矛盾，放大既有优势。

| 计策 | 商业化算子 | 典型用法 |
| --- | --- | --- |
| 瞒天过海 | 在常规动作中隐藏真正意图 | 用例行沟通、标准流程、公开叙事掩护关键资源调度。 |
| 围魏救赵 | 不正面硬刚，攻击对方依赖点 | 竞品低价压制时，转向质保、风险、交付稳定性等其薄弱维度。 |
| 借刀杀人 | 借第三方规则、平台、客户声音形成压力 | 用行业标准、媒体评测、客户案例、合规要求改变对方成本结构。 |
| 以逸待劳 | 保持资源耐心，让对方消耗 | 不急于降价或承诺，设置验证环节，让对方暴露真实需求。 |
| 趁火打劫 | 在对手混乱时获取窗口利益 | 对方供应中断、舆情波动、组织调整时快速抢客户、渠道或议价权。 |
| 声东击西 | 制造注意力偏移，实攻另一目标 | 表面谈价格，实际推进付款条款、数据权、排他期或渠道控制。 |

## 敌战计：均势局创造信息差

Core logic: 掩饰意图，以局部代价换全局主动。

| 计策 | 商业化算子 | 典型用法 |
| --- | --- | --- |
| 无中生有 | 先制造可被验证的叙事，再转为真实筹码 | 用试点、白皮书、预约需求、候选方案形成市场预期。 |
| 暗度陈仓 | 表面推进常规事项，真实推进关键控制点 | 合同中以服务审查、数据接口、质量稽核嵌入长期治理权。 |
| 隔岸观火 | 延迟介入，让冲突双方自耗 | 在竞品与客户价格拉扯中观察真实预算、痛点和承诺能力。 |
| 笑里藏刀 | 友好表层下设置硬边界 | 保持合作语气，同时用条款、里程碑、验收标准锁定风险。 |
| 李代桃僵 | 牺牲低价值资源保核心资源 | 放弃小折扣、小赠品或非关键权益，换取付款、数据、排他或长期订单。 |
| 顺手牵羊 | 抓住低成本机会顺势获利 | 借竞品热度、客户提问、行业新闻获取流量、线索或谈判信息。 |

## 攻战计：僵局中主动破防

Core logic: 制造诱饵，调动对方核心资源，暴露关键约束。

| 计策 | 商业化算子 | 典型用法 |
| --- | --- | --- |
| 打草惊蛇 | 小动作测试对方反应 | 抛出非核心条款、试探报价或限量条件，观察真实底线。 |
| 借尸还魂 | 借旧资产、旧渠道、旧案例重启价值 | 复用沉默客户、旧内容、旧技术方案进入新市场叙事。 |
| 调虎离山 | 让对方离开优势位置 | 把竞争从价格拉到交付、认证、售后、定制能力等对方弱项。 |
| 欲擒故纵 | 先放松控制，让对方增加投入 | 不急着成交，给对方试用、方案、样品或选择空间，换取更深需求暴露。 |
| 抛砖引玉 | 用小价值诱发大信息或大资源 | 用免费诊断、样品、问题清单换取客户预算、流程、决策链信息。 |
| 擒贼擒王 | 找到真正决策点并集中突破 | 识别预算人、法务红线、技术负责人或舆论源头，减少外围消耗。 |

## 混战计：乱局中重构规则

Core logic: 切断对方供应链、退路或默认规则，重建议价场。

| 计策 | 商业化算子 | 典型用法 |
| --- | --- | --- |
| 釜底抽薪 | 处理根因而非表象 | 切断对方低价优势的前提：质量疑虑、售后成本、合规风险、供货不稳。 |
| 混水摸鱼 | 在信息混乱时获取被低估资源 | 行业波动时抢低价广告位、人才、渠道、库存或客户注意力。 |
| 金蝉脱壳 | 保留表面连续性，转移核心资产 | 退出低利润渠道，同时保住客户数据、品牌信誉、核心团队。 |
| 关门捉贼 | 限定范围后集中解决 | 把谈判、投诉、竞品比较限制在可控议题和时间窗口内。 |
| 远交近攻 | 与远端盟友合作，压缩近端对手 | 联合平台、上游、行业协会或跨区伙伴制衡本地竞品。 |
| 假道伐虢 | 借合作路径进入关键位置 | 通过联合项目、代理合作、试点集成进入客户或渠道核心流程。 |

## 并战计：消耗战中夺取主导权

Core logic: 控制沉没成本，渗透接管，重定义角色。

| 计策 | 商业化算子 | 典型用法 |
| --- | --- | --- |
| 偷梁换柱 | 替换关键评价标准或底层依赖 | 从单价比较改为总拥有成本、生命周期、交付确定性、数据沉淀。 |
| 指桑骂槐 | 通过旁敲侧击传达边界 | 不直接指责客户或伙伴，而用行业案例说明风险和红线。 |
| 假痴不癫 | 表面迟钝，实际稳住节奏 | 对压价、诱导承诺、模糊条款不正面反击，先收集信息和拉长节奏。 |
| 上屋抽梯 | 先让对方进入高投入状态，再锁条件 | 提供初步方案、样品、试点框架后，再明确付款、排他、验收或数据条件。 |
| 树上开花 | 借外部信号放大实力 | 用认证、背书、案例、合作伙伴、媒体露出增强谈判声势。 |
| 反客为主 | 从被动供应方转为规则制定方 | 提供模板、流程、指标、审查机制，逐步掌控合作节奏。 |

## 败战计：逆风局止损与反杀

Core logic: 制造内部瓦解，果断切割重组，保留可再战资产。

| 计策 | 商业化算子 | 典型用法 |
| --- | --- | --- |
| 美人计 | 用高吸引力资源改变决策权重 | 用设计、品牌、体验、关键人才、样板案例改变客户关注点。 |
| 空城计 | 在资源不足时制造确定性与不可测性 | 以强流程、明确边界、公开节奏稳住对方，不暴露短板。 |
| 反间计 | 让对方内部信息和利益冲突显性化 | 用公开问题、对比表、角色分工让其决策链内部自行校验。 |
| 苦肉计 | 承受可控小损失换取信任或反转 | 主动让利、试点担责、公开修正失误，以换取长期信任。 |
| 连环计 | 多个动作互相锁定，形成连续约束 | 内容、合同、渠道、付款、数据接口多点组合，降低对方逃逸空间。 |
| 走为上 | 主动退出不可赢局面 | 切断坏客户、低质渠道、失控合同或错误舆论场，保留现金和信誉。 |
